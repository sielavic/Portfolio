Обо мне

Здравствуйте, меня зовут Станислав. Я фрилансер. Приглашаю вас посмотреть мои работы и спасибо за симпатию к проектам.
В 2016 году на фрилансе выполнял постоянные заказы челябинской компании по наполнению дополнительных сайтов cms с seo.
Впервые столкнулся с web разработкой в 2012 году. Делал сайты совместно с веб дизайнером не на cms и собирал без чьей либо помощи на cms. Использовал компании Google Adwords для продвижения. Также переделывал игровые сервера с привязкой к сайтам. Там использовал js, php, MySql.
Основные навыки:
- навыки стилизации в минимализме, американском бизнес стиле, стиле веб 3.0, промостиле
- верстка с учетом 12 колонок в модульной сетке с избежанием двухколонников и трехколонников, использование нестандартных вариантов модульных сеток
- GIT
- HTML5
- CSS3
- CMS Opencart, 1CBitrix
- Boostrap 4
- jQuery
- асинхронный не типизированный линейный JavaScript(ES6) с использованием антивложенностей(исключение адов колбеков) путем промисификации.
- React, React Native, React Navigation (v4, v5), Expo - Node.js и фреймворк Express
- Сборщик Webpack
- Php( v 7.x) и фреймворк Yii2, API Bitrix Framework, GII и миграции
- Интеграция дизайна в cms
- Создание собственной логики в cms
- Шаблонизатор twig
- MySql, SQlite, NoSql ,MongoDB, AWS, Real time Firebase для чатов, FirebaseAuth
- Работа консолью ISPmanager подключение домена, полноценное развертывание на хостинге, настройка https и SSL.
- unix команды такие как cd touch pwd ls -la доведены до автоматизма.
- Пользуюсь Microsoft Visual Studio и для java использую NetBeans.
- На английском языке пишу разговариваю, так как некоторые друзья совершенно не знают русского языка.
- верстка макетов с помощью Avacode и аналогов.
- изменение модели MVC в cms, переопределение
глобальных настроек бутстрап, jQuery.
Личные качества:
- Настойчивость
- Стрессоустойчивость
- Гибкость ума
- Скорость мышления
- Коммуникабельность
